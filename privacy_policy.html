ReminDo

プライバシーポリシー

最終更新: 2026年6月6日

1. はじめに

ReminDo（以下「本アプリ」）は、NARUKAMI AKUT が提供する家族向けリマインドアプリです。本ポリシーは、本アプリがどのような情報を収集し、どのように使用するかを説明します。

2. 収集する情報

情報の種類用途外部送信
メールアドレス アカウント登録・認証 あり（Google / Firebase）
ユーザー ID デバイス間のデータ同期 あり（Google / Firebase）
タスク・プロファイルデータ アプリ機能・デバイス間同期 あり（Google / Firebase）
アプリの操作ログ 機能改善のための利用状況把握（Firebase Analytics） あり（Google）
クラッシュ情報 不具合の検出・修正（Firebase Crashlytics） あり（Google）
広告識別子 広告配信（Google AdMob） あり（Google）
カレンダーデータ Google カレンダーとのタスク同期（連携オン時のみ） あり（Google）

3. 第三者サービス

本アプリは以下の Google サービスを利用しています。各サービスの個人情報取り扱いについては、Google のプライバシーポリシーをご確認ください。

  • Firebase Authentication — アカウント管理・認証
  • Cloud Firestore — デバイス間データ同期
  • Firebase Analytics — 匿名の利用統計
  • Firebase Crashlytics — クラッシュレポート収集
  • Google AdMob — 広告配信（無料プランのみ）
  • Google Calendar API — カレンダー同期（ユーザーが連携をオンにした場合のみ）

Google プライバシーポリシー →

4. アカウントとデータの削除

本アプリ内からアカウントとすべてのデータを削除できます。

アプリを起動 → 設定 → アカウント → アカウントを削除
削除すると、サーバー上のすべてのデータおよびアカウント情報が完全に削除されます。この操作は取り消せません。

削除のご依頼やご不明点は、下記連絡先までお問い合わせください。

5. 子どものプライバシー

本アプリは家族向けに設計されており、13歳未満のお子様から意図的に個人情報を収集することはありません。保護者の管理のもとでご利用ください。

6. ポリシーの変更

本ポリシーは予告なく変更される場合があります。変更後のポリシーはこのページに掲載し、最終更新日を更新します。

7. お問い合わせ

プライバシーに関するご質問は下記までご連絡ください。

Privacy Policy

Last updated: June 6, 2026

1. Overview

ReminDo is a family reminder app developed by NARUKAMI AKUT. This policy explains what information the app collects and how it is used.

2. Information We Collect

Data TypePurposeTransmitted
Email address Account registration and authentication Yes (Google / Firebase)
User ID Cross-device data sync Yes (Google / Firebase)
Tasks and profile data App functionality and cross-device sync Yes (Google / Firebase)
App usage logs Feature improvement (Firebase Analytics) Yes (Google)
Crash reports Bug detection and fixes (Firebase Crashlytics) Yes (Google)
Advertising identifier Ad delivery (Google AdMob) Yes (Google)
Calendar data Task sync with Google Calendar (only when integration is enabled) Yes (Google)

3. Third-Party Services

The app uses the following Google services. Please refer to Google's Privacy Policy for details on how they handle data.

  • Firebase Authentication — account management and authentication
  • Cloud Firestore — cross-device data sync
  • Firebase Analytics — anonymous usage statistics
  • Firebase Crashlytics — crash report collection
  • Google AdMob — ad delivery (free plan only)
  • Google Calendar API — calendar sync (only when the user enables the integration)

Google Privacy Policy →

4. Account and Data Deletion

You can delete your account and all associated data from within the app.

Open the app → Settings → Account → Delete Account
All data stored on our servers will be permanently deleted. This action cannot be undone.

For deletion requests or questions, please contact us at the address below.

5. Children's Privacy

This app is designed for families and does not intentionally collect personal information from children under 13. Please use the app under parental supervision.

We may update this policy at any time. Changes will be posted on this page with an updated date.
